# hyperspectR: Hyperspectral Imaging Analysis for Biomedical Applications

Read, process, analyze, and visualize hyperspectral imaging (HSI) data
from Cubert snapshot cameras and standard ENVI/TIFF formats. Provides
calibration workflows (dark correction, white reference normalization),
spectral preprocessing (Savitzky-Golay smoothing, SNV, MSC,
derivatives), biomedical tissue indices (tissue oxygenation StO2,
perfusion NPI, hemoglobin THI, water content TWI), dimensionality
reduction (PCA, MNF), pixel-wise classification (SAM, SVM, random
forest), and spectral unmixing via Beer-Lambert chromophore fitting.
Includes an interactive 'shiny' application for clinical hyperspectral
image exploration. Designed for intraoperative imaging, soft tissue
diagnostics, and perfusion monitoring workflows.
